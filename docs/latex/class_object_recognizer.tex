\subsection{Detailed Description}
Class used to recognize objects in a video stream. 

This class manages a thread taking a frame from an input stream, analyzing it to find some kind of 3\-D object and then updating the data used by an Object\-Tracker to track it. \begin{DoxyAuthor}{Author}
